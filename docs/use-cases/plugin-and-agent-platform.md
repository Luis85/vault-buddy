# Plugin Platform & Specialized AI Agents

> External integrations (Git, GitHub, Jira, Terminal, Browser, Filesystem, Calendar, Email, Cursor, Claude, MCP, Voice, Home Assistant, ...) as installable Plugins registering new capabilities; specialized AI Agents (Research, Documentation, Writing, Planning, Architecture, Meeting, Developer, Review, Automation) orchestrated by Vault Buddy as Phase 4/5 of the roadmap.

## Source

Main PRD, [§12 Plugin Architecture](../PRD%20-%20Product%20Vision.md) and [§13 Agent Architecture](../PRD%20-%20Product%20Vision.md); [§18 Roadmap → Phase 4 — Platform](../PRD%20-%20Product%20Vision.md) (Plugin SDK, Plugin Marketplace, Agent SDK, Workflow Marketplace, Community Extensions) and [Phase 5 — Knowledge Operating System](../PRD%20-%20Product%20Vision.md) (multi-agent orchestration). AI Platform PRD, [Plugin Platform](../prds/ai-platform.md#plugin-platform) section and [Product Roadmap → Phase 3/4](../prds/ai-platform.md).

## Status: Not started

No plugin loading mechanism, plugin manifest format, or agent-orchestration code exists. This is the furthest-out item in the roadmap (Phase 4–5 of the main PRD) and presupposes [MCP Server & Runtime](mcp-server-and-runtime.md) as a foundation.
