# Buddy sprite attribution

The pixel-art character sprites in this directory (`*-idle.png`, `*-run.png`)
are cropped from **Dungeon Tileset II v1.7** by **0x72**:

- Source: <https://0x72.itch.io/dungeontileset-ii>
- License: **CC0** (Creative Commons Zero) — the author states
  "You can use this tileset for whatever you like."

Each file is a horizontal 4-frame strip (16×28 px per frame) assembled from
the pack's individual frame PNGs:

| File | Pack character |
| --- | --- |
| `knight-*.png` | `knight_m` |
| `wizard-*.png` | `wizzard_f` |
| `elf-*.png` | `elf_f` |
| `dwarf-*.png` | `dwarf_m` |
| `lizard-*.png` | `lizard_m` |

Attribution is not required by CC0, but credit goes to 0x72 — thank you for
the wonderful sprites!
